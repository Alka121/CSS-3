CSS-3 RADIAL GRADIENTS...!

In a radial gradient color emerge from a single point and smoothely spread outword in a circular or eliptical shap rather than fanding one color to other in the single direction as with linear gradient the basic synatax of creating a radial gradient can be given with

Syntax Radial gradient (shap size at position, color-stop1,color-stop2...)

the argument of color gradient () function has the following meaning.

  1. position-specifies the starting point of gradient which be specified in unit(px,em or percentage)or keyword (left, bottom,etc)
  2. shape-specifies the shape of gradientsending shape.it can be circle or ellipse.
  3. size-specifies the size of gradientsending ending shape.the defalt is furthest side.
